Project Cerebro: Distributed Task Scheduler

A fault-tolerant distributed task scheduling system designed for high throughput. Implements Raft consensus for leader election and log replication, ensuring no single point of failure.

Tech Stack: Go, gRPC, Protocol Buffers, Raft, Docker, Prometheus.

KubeWatch: Kubernetes Event Monitor

A real-time monitoring tool for Kubernetes clusters that streams events and provides insights into cluster state changes. Features a web-based dashboard for visualization.

Tech Stack: Python, FastAPI, WebSockets, React, Kubernetes API.

GitHub Repo

GeoCache: Distributed Caching Service

A simple distributed caching service with support for sharding and data replication. Explores concepts like consistent hashing and cache eviction policies.

Tech Stack: Java, Spring Boot, Redis (for metadata), gRPC.

GitHub Repo

StreamFlow: Event Processing Framework

A lightweight framework for building and managing event-driven data processing pipelines. Inspired by Apache Flink and Kafka Streams, but simplified for smaller scale applications.

Tech Stack: Scala, Akka Streams, Kafka.
